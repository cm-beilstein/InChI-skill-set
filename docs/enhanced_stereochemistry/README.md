# Enhanced Stereochemistry Research

Research report on enhanced stereochemistry detection and encoding across major cheminformatics platforms.

## Contents

- `report.md` - Full research report

## Covered Topics

- Theoretical foundations (ABS, OR, AND stereogroups)
- File format encodings (V3000 molfile, CXSMILES)
- Software implementations:
  - BIOVIA/Dassault Systèmes (Draw)
  - ChemAxon (MarvinSketch, JChem, Marvin JS)
  - RDKit
  - ChemDraw
- Canonicalization approaches
- Atropisomer integration

## Key Findings

1. Enhanced stereochemistry is the industry standard for uncertain stereochemistry
2. V3000 format required for full support; CXSMILES most readable
3. RDKit has the most active open-source development
4. ChemAxon implementation is mature but some features deprecated
5. Canonicalization is essential for database registration
